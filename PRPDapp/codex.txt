PRPD — GUI Unificada (Windows 11 + Python 3.13)Este proyecto ofrece una única interfaz gráfica (PySide6) para análisis PRPD con filtrado, vistas de diagnóstico y exportaciones reproducibles.RequisitosPython 3.13 (Windows 11)Paquetes: PySide6, numpy, matplotlib, scikit‑learn (instala con `pip install -r PRPDapp/requirements.txt` si fuera necesario)Cómo ejecutarOpción recomendada (desde el repo):Doble clic: `PRPDapp\run_gui.bat`Consola: `cd <raiz del repo>` y `python -m PRPDapp.main`(Compat) Variantes reenvían a la GUI unificada:`python -m PRPDapp.main_utf8``python -m PRPDapp.psg_main`Flujo de trabajo en la GUICargar archivo (CSV o XML)Fase: auto (0/120/240) o fija (selector)Filtro ruido: `Filtro: S1 Weak | S2 Strong`Vista:Probabilidades (ANN/heurística)ANGPD (angpd y n_angpd; curvas)Nubes (clustering fino con centroides)ANN: botón “Cargar ANN” (opcional), soporta `.pkl`/`.joblib`. Si existe `models/feature_order.json`, se usa para alinear el vector de entrada.Baseline: botón “Reset baseline” crea/actualiza bandas ±20% (solo UI).Exportar (automático al “Procesar”; PDF opcional en implementaciones futuras)Exportaciones (out\reports)Para cada archivo `<stem>` procesado:ANGPD: `<stem>_angpd.csv` (phi_deg, angpd, n_angpd) y `<stem>_angpd.png`Nubes:`<stem>_clouds_raw.csv` (id,count,frac,phase_mean,y_mean)`<stem>_clouds_combined.csv` (ids,count,phase_mean,y_mean)`<stem>_clouds_selected.csv` (id,count,frac,phase_mean,y_mean)`<stem>_clouds.png`Métricas: `<stem>_metrics.csv` (predicted,severity,p95_amp,dens,R_phase,std_circ_deg,phase_offset,filter_level)Baseline: `<stem>_baseline.json` (se crea si no existe; usado para bandas ±20% en UI)BatchBotón “Procesar carpeta” en la GUI (usa el filtro y la fase seleccionados). Exporta reportes en `out\batch\...` y resumen JSON/Texto.Solución de problemas“ModuleNotFoundError: No module named 'PRPDapp'”Ejecuta siempre en modo módulo desde la raíz del repo: `python -m PRPDapp.main` o usa `PRPDapp\run_gui.bat`.Falta PySide6 / paquetes: instala con `pip install -r PRPDapp/requirements.txt`.Codificación extraña en textos: la GUI unificada usa UTF‑8; si ejecutaste `main_utf8.py`, ya redirige a la unificada.Estado del proyectoInterfaz única: `PRPDapp.main`Compatibilidad: `PRPDapp.main_utf8` y `PRPDapp.psg_main` se mantienen, pero reenvían/conviven sin duplicar funcionalidades.Núcleo: `PRPDapp.prpd_core`Nubes (S3–S5): `PRPDapp.clouds`NotasLos CSV/PNG de export se sobrescriben en cada procesamiento. Las escrituras son robustas y se crean carpetas si faltan.El baseline es por archivo (`<stem>_baseline.json`). Usar “Reset baseline” para re‑anclar bandas ±20% con el valor actual.
